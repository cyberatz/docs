== Getting Started

To get started with our project, follow these simple steps:

[.procedure]
. Clone the repository
+
[source,bash]
----
git clone https://github.com/your-org/your-repo.git
----

. Install dependencies
+
[source,bash]
----
npm install
----

. Start the development server
+
[source,bash]
----
npm run dev
----
